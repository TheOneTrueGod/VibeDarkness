# Writing style — worlds

**Read** `STYLE.md` here before inventing place names, local slang, tech level, or metaphysics for a **new or returning** setting.

Keep **world bibles** thematic; do not list every map segment or mission ID here—use the **map-segments** and **missions** skills for structure, and this doc for **voice of place**.
