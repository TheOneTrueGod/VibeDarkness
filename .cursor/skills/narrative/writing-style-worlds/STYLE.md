# Worlds writing — style template

Mark sections with **`[Canon]` / `[Draft]` / `[Open]`** per `narrative-hub`.

## World sheet pattern [Draft]

For each major place the story visits, maintain a **short** block (themes, not catalogs):

- **Name + one-line pitch**
- **Sensory defaults:** weather, soundscape, architecture shorthand agents can reuse.
- **Moral / thematic axis:** what conflicts *feel* like here.
- **Continuity boundaries:** what this world **does not** import from others (tech, cosmology, tone).
- **Stability:** suffix on the **world section heading**.
