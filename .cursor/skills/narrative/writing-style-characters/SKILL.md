# Writing style — characters

**Read** `STYLE.md` here before writing lines for **named** characters that appear in more than one place.

This is **not** a cast list: use **voice sheets** (patterns per character) and point to in-game sources for lines. Full rosters belong in campaign or game data, not here.

If **`STYLE.md` has no voice sheet** for a named character you are about to write for: follow the **missing voice sheet** section in `STYLE.md` (search repo sources, then **stop and ask**, with **proposals** grounded in what you found). **Do not** treat a missing sheet as permission to invent a **fixed canon** voice with no user check.
