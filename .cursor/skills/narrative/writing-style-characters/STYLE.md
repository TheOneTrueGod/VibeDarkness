## When a named character has no voice sheet here [Draft]

Applies when you need **distinct** voice or biography for a **named** character and `STYLE.md` has **no** matching voice-sheet block (or only the generic placeholder).

1. **Mine the repo first** — campaign, missions, UI copy, bios, prior dialogue, and (if setting-relevant) `lore-primer/LORE.md`. Prefer **evidence** over invention.
2. **Stop and ask the user** before shipping substantial new characterization. Do **not** treat a missing sheet as permission to invent a **fixed canon** voice with no check.
3. In the same message, **propose** a short candidate voice sheet (name + role, speech markers, taboos) framed explicitly as **inference from available material** where you have any, or as **open questions** where you do not. Label proposals **`[Draft]`** / **`[Open]`** as appropriate; do not present guesses as **`[Canon]`**.
4. After the user confirms (or adjusts), you may add a **concise** voice-sheet block here or author lines in the **source** files this doc points to—without pasting long catalogs into skills (see `narrative-hub`).

## Voice sheet pattern (per recurring character) [Draft]

Use a **repeatable block** for each important character (keep each block short):

- **Name + role:** one line.
- **Speech markers:** vocabulary, rhythm, verbal tics (not long samples).
- **Taboos:** what they would not say or do in copy.
- **Stability:** tag the **block heading** with `[Canon]` / `[Draft]` / `[Open]` for that character’s sheet.
